# Just-Cook-It

A web app that helps users decide what to cook based on
ingredients, dietary preferences, and available time.

## Features
- Ingredient-based recipe suggestions
- Dietary restriction filtering
- Day-by-day cooking time preferences

## Tech Stack
- HTML, CSS, JavaScript
- Spoonacular API (planned)

## How to Run
1. Clone the repo
2. Open `index.html` in browser
